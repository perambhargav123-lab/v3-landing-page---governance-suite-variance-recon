# PBV Brain Showcase

Static client showcase for PBV Finance Brain.

## What This Is

This is a public-facing landing/demo page for a controlled AI finance workflow system. It shows:

- 26-claw finance architecture.
- Phase 1 evidence chain.
- Source-to-governance pipeline.
- Evidence-pack model.
- Pricing and pilot offer.

## Important Claim Boundary

This is a showcase for controlled pilots. It does not claim production-live status, autonomous ERP posting, payment execution, tax filing, external sending, or replacement of human finance review.

## Deploy

This folder is deployable as a static Vercel site. `index.html` is the entry point.

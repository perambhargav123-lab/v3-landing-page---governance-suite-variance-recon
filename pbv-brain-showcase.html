AI cannot self-approve

CFO questions

Objections answered before the call.

Is this SaaS?

Not yet. It is a productized pilot system and evidence workflow. SaaS-style runtime controls come later after pilots prove repeatability.

Do we need SAP integration?

No. Phase 1 uses manual or sanitized exports. Scheduled/API integration only happens after IT approval and repeated clean runs.

Will AI see confidential raw data?

The default pilot can use sanitized files. For real data, source handling is agreed under NDA and client approval before any run.

Can this replace my finance team?

No. It is designed to make controllers and FP&A faster, better evidenced, and more consistent. Humans remain reviewers and decision owners.

What happens if data is messy?

P0 does not hide that. It returns Approved, Diagnostic Only, or Blocked with exact exceptions and downstream limits.

What do we need for a first call?

A sample report pack, one sanitized export if available, current close/variance pain points, and the person who would review outputs.
